# Privacy Policy for Molecules Daily Pinterest Automator

**Effective Date:** March 29, 2026

This application ("Molecules Daily Pinterest Automator") is a private automation tool used to share educational chemical information from the Molecules Daily Telegram channel to Pinterest.

### 1. Data Collection
We do not collect, store, or share any personal data from Pinterest users. The application only accesses the user's own Pinterest account (via an authorized Access Token) to post content to a specific, user-defined board.

### 2. Use of Pinterest Data
The application uses the Pinterest API v5 to:
- Create Pins (images and descriptions) on a specified board.
- We do not read your profile data, followers, or other private information.

### 3. Third-Party Services
We use the PubChem API (National Library of Medicine) to retrieve public chemical structure images and data. No user data is sent to PubChem.

### 4. Data Security
Your Pinterest Access Token is stored securely as a GitHub Secret and is never exposed in the code or logs.

### 5. Contact
If you have questions, please contact the repository owner via GitHub.
